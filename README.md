# Learning HTML & CSS - All Pages (2022)

This repository contains a collection of HTML and CSS code examples, practice exercises, and small project pages created in 2022 while learning web development basics.

## Contents
The repository includes:
- **Basic HTML Elements**: Formatting tags (bold, italic, underline, superscript, subscript), headings, paragraphs, and list styles (ordered, unordered, nested, description lists).
- **Media Integration**: Implementation of images, audio (`.mp3`), and video (`.mp4`) tags.
- **Tables**: Various table layouts including student marks, timetables, and usage of `colspan` and `rowspan`.
- **Forms**: Form controls, custom input validations, registration forms, fieldsets, and other interactive elements.
- **Frames**: Basic HTML frame tags.
- **CSS Styles**: Inline, internal, and external stylesheets (including simple CSS layout webpages like Black Goose and custom menu designs).

## How to Use
You can open any of the `.html` files in a standard web browser (such as Google Chrome, Mozilla Firefox, or Microsoft Edge) to view the rendered outputs and inspect the underlying code.
